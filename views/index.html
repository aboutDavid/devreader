DevReader

The RSS reader built for devs using HalfmoonCSS for the UI, webpack for the static webpages, and express.js as the backend.

On the sidebar, you can view your feeds. This is a public instance which gives you some feeds by default. There is a light mode toggle because it uses dark mode by default. Feeds update every 10 minutes and it is hosted on Glitch!

It's also mobile friendly :)

Tips:

Press shift + f to toggle the sidebar
